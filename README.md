# Quick Commerce Data Scraping & Analysis

##  Project Overview

This project focuses on collecting, cleaning, and structuring product-level data from quick commerce platforms to analyze brand availability, pricing, and stock patterns.

## 🎯 Objective

To build a structured and analysis-ready dataset of top haircare brands across multiple platforms and locations.

## 📍 Scope

* Platforms: Blinkit, Zepto, Swiggy Instamart
* Locations: Bangalore (5 pin codes)
* Brands: Top haircare brands

## 📂 Dataset Structure

### Sheet 1: Raw Data

* Brand Name
* Product Name
* Platform
* Pin Code
* MRP
* Selling Price
* Discount (%)
* Stock Status
* Delivery Time

### Sheet 2: Brand Summary

* Brand availability across platforms
* Average discount per brand
* Out-of-stock count

## 🧹 Data Collection & Cleaning

The dataset required **significant effort to collect and preprocess**:

* Manual data extraction across multiple platforms
* Handling inconsistent formats and missing values
* Converting discount fields from text (e.g., "13%OFF") to numeric
* Standardizing brand names and removing inconsistencies

## 📊 Key Outcome

The final dataset is:

* Clean and well-structured
* Consistent across all fields
* **Ready for analysis, insight generation, and dashboard creation**
